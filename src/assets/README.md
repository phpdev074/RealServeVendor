# RealServ Brand Assets

This folder contains RealServ brand assets for use across the application.

## Files:

- `realserv-logo.png` - RealServ logo (R icon in rounded square) - Use for small spaces
- `realserv-wordmark.png` - RealServ wordmark (Full "RealServ" text) - Use for headers and branding

## Usage Guidelines:

### Logo (Icon "R"):
- Sidebar branding
- Mobile navigation
- Favicon
- Small UI elements
- Loading states

### Wordmark (Full "RealServ" text):
- Login/signup pages
- Main navigation headers
- Landing pages
- Email templates
- Large branding sections
